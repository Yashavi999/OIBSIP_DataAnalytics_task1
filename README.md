# OIBSIP_DataAnalytics_task1
Exploratory Data Analysis (EDA) on Retail Sales Data
**Retail Sales Data Exploration Project**
This project conducts **Exploratory Data Analysis (EDA)** on retail transaction data to uncover customer buying behavior, sales patterns, and meaningful business insights.

**Dataset**
customer_shopping_data.csv
menu.csv (optional dataset)

**Objectives**
Perform data cleaning and data preparation
Examine summary and descriptive statistics
Evaluate sales patterns across different time periods
Analyze customer demographics and product category performance
Visualize findings using charts, graphs, and heatmaps
Generate strategic business insights and recommendations

**Tools & Technologies**
Python
Pandas
NumPy
Matplotlib
Seaborn
Google Colab

**Conclusion**
This analysis enables businesses to better understand customer purchasing habits and supports informed, data-driven business strategies.
